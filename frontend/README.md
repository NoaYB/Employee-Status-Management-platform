# Employee Management Frontend

A React application for managing employee cards. Built with Vite and TypeScript.

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the dev server:
   ```bash
   npm run dev
   ```

3. Open `http://localhost:5173` in your browser.

## Features

- **Employee List**: Displays all employees with avatars and status.
- **Create Modal**: Allows adding new employees.
- **Status Update**: Dropdown on each card to change status instantly.
- **Delete**: Confirmation dialog before removing an employee.
- **Responsive**: Modern CSS (Cream/Minimalist theme).

## Configuration

Ideally connects to `http://localhost:3000`. 
Check `src/App.tsx` if you need to change the API base URL.
